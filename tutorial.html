Create Project

    
        $: mkdir my_app
Create package.json
    $: npm init -y
Install apollo-server-express & mongoose
    $: npm install apollo-server-express mongoose
Install nodemon
    $: npm install nodemon --save-dev
Create project files
        $: touch server.js
        $: mkdir graphql model
        $: cd graphql
        $: mkdir resolvers
        $: touch typeDefs.js
        $: cd resolvers
        $: touch index.js
        $: touch sales.js
        $: cd ..
        $: cd ..
        $: cd model
        $: touch User.js
Development startup script

Add the following key value to your script section of your package.json

    “dev” : “nodemon server.js”

It should now look like the following

    
        "scripts": {
            "test": "echo \"Error: no test specified\" && exit 1",
            "dev": "nodemon server.js" },

Create your server - server.js

        const express = require('express');
        const { ApolloServer , gql } = require('apollo-server-express');
        const mongoose = require('mongoose');
        
        //Enter your actual connection string here
        const db_con_string = "enter your connection string here"
        
        const PORT = process.env.PORT || 8080;
        const app = express();
        mongoose.connect(db_con_string, { useNewUrlParser : true })
            .then(() => {
                console.log("MongoDB Connected");
                return app.listen({ port : PORT })
            })
            then((res) => {
                console.log(`Application is listening on port ${PORT}. Browse to http://localhost:${PORT}`)
            })
            .catch((err) => {
                console.log(err)
            });

Now if you run your development script (npm run dev) your server should start. You can navigate to the URL listed in your console (localhost:8080)
