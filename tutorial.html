Download The Project Files

If you dont have github installed you can download the file as a zip from the github repo. Otherwise clone this project

Download Package Files

Navigate using your console to the project directory and run npm install

Database Connection Details

Once you've downloaded the project open your db_config.js file and modify it to use these values
const db_username = "tutorialuser";
const db_password = "7CQLhe6Bfx6tUCXC";
const db_database = "sample_supplies";

Test the application

Once you've modified the above values you should be able to run npm run dev and the application will boot. You should see a message in your console saying
MongoDB Connected
Application is listening on port 8080. Browse to http://localhost:8080/graphql

Open your browser and navigate to localhost:8080/graphql. A window should load with a panel on the left and a panel on the right.
In the left hand panel copy and paste the following.
query { getAllSales { _id saleDate storeLocation couponUsed purchaseMethod items { name tags price quantity } customer { gender age email satisfaction } } }
Hit the prettify button to format the text into something more readable. Now hit the play button in the middle of the screen. This query may take a little while to load as you're fetching every entry in the collection (/database).

Now try changing the query to getSaleById. You'll need an ID to query the database with. Heres a random one 5bd761dcae323e45a93ccff0. after the word getSaleById put round parentheses and press LCtrl & Space. It should auto fill the word id. Now if you put your ID in quotes after the letters id like this id:"5bd761dcae323e45a93ccff0" and hit search you should get back only that result.
